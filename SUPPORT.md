# Support

Use GitHub Issues for reproducible bugs and setup problems. Include macOS,
Python and Node versions, the failing command, sanitized output, and whether
`weekly-report doctor --offline` and `--live` pass.

Never attach `secrets.env`, full config files, LINE IDs, Cloudflare IDs, signed
media URLs, private PDFs, or generated reports. Security issues belong in the
private channel described by `SECURITY.md`.
